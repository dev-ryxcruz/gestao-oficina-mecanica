# Sistema de Gestão para Oficina Mecânica

## 👤 Autor
* **Ryan Cristian**

## 📝 Resumo do Projeto
Este projeto visa organizar o fluxo de atendimento de oficinas, substituindo controles manuais por um sistema digital que gerencia Ordens de Serviço (OS) e mantém um histórico inteligente de cada veículo.

## 🛠 Tecnologias
* **Linguagem:** C#
